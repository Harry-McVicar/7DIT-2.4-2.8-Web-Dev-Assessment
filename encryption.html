Encryption

Encryption is the process by which data, also known as plaintext, is converted into unreadable ciphertext so that it is protected from potentially being intercepted by hackers. This is achieved through the use of a key which encrypts and decrypts data, preventing it from being intercepted inbetween the sender and the reciever.

What is encyption and why is it so important?

Encryption is important as it prevents private data from being accessible to hackers, who could potentially use your personal details to blackmail or target you with phishing schemes. Some encryption methods are more breakable than others, such as caesar cipher, but the algorithms used in modern society are typically very sturdy. The public-private key method which is primarily used today is extremely hard to break, but not impossible.

The key distribution problem

The fundamental issue of the key distribution problem is that, while the ciphertext can be sent safely across an unsecure network, the key cannot. Single key encryption, also known as symmetric encryption, uses the same key to encrypt the message as it uses to decrypt the message. This means that the user and the sender must each know the key in order to send and read one another’s messages. Their data cannot be intercepted by hackers as only they know the secret key. Public key encryption, also called asymmetric encryption, uses a one way function / algorithm which enables people to send encrypted messages. This method encrypts the sender’s plain text with the receiver’s public key. The cipher text can therefore only be decrypted using the receiver’s private key, and can not be intercepted by hackers, solving the key distribution problem.
